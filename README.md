# Spacecraft

An on-rails space shooter built with the [Nightshade](https://github.com/matthewjberger/nightshade) game engine.

Fly a Spitfire through deep space on a rail: steer across the play field, bank and
barrel roll, blast asteroids with the wing turrets, and thread the glowing gates.

## Controls

| Action | Keyboard | Gamepad |
|--------|----------|---------|
| Steer | WASD / Arrow keys | Left stick |
| Fire | Space | A / X |
| Barrel roll | Q / E | L1 / R1 |
| Boost / Brake | Shift / Ctrl | R2 / L2 |
| Quit | Esc | - |

## Run it

```bash
# native
just run

# in the browser (WebGPU)
just run-wasm
```

> WebGPU browsers: any Chromium based browser (Chrome, Edge, Brave, Vivaldi) or
> Firefox 141+.
